# Codex Task: Improve Cultural Insights for Classroom Guide

## Context

stool is a React+Vite SPA for international school teachers. The **Classroom Guide** tab (`src/components/tabs/ClassroomGuide.jsx`) helps teachers understand student behavior through a cultural lens. It currently uses Hofstede's 6 cultural dimensions (PDI, IDV, MAS, UAI, LTO, IVR) stored in `src/data/hofstede.js` to generate country-specific classroom advice.

The guide has two modes:
1. **Country Guide** — select a country, see 5 category summaries (Participation, Communication, Learning styles, Parent relationships, Relationships) plus expandable behavior cards
2. **Explore by Behavior** — select classroom behaviors you're seeing, get matched countries

Behavior data lives in `src/data/faqData.js` (12 behaviors currently, each with `why`, `respond`, `research`, `current_context()` function, and destination advice).

Category summaries are generated by threshold functions in `CATEGORY_CONFIG` inside `ClassroomGuide.jsx`.

**Important constraint**: A caveat box at the top explains these are national-level tendencies, not predictions about individuals. Abena (a 10-year veteran in Malaysia) pointed out that student silence there comes more from previous school system experience (punishment for speaking up) than from group culture per se. All new content must respect this nuance — prior schooling experience and individual variation matter as much as national culture.

## What needs improving

Current count: 12. Target: 20–25. Each behavior object must follow this exact schema:

```js
{
  id: 'f13',                    // sequential
  behavior: 'Short description of what the teacher sees',
  dims: ['Dimension name'],     // human-readable Hofstede dimension(s) involved
  category: 'participation',    // one of: participation, communication, learning, parents, relationships
  dim_key: 'PDI',               // primary Hofstede dimension abbreviation
  dest_key: 'PDI',              // dimension used for destination comparison
  current_context: (cc, hc) => {
    // cc = country name string, hc = Hofstede array [PDI,IDV,MAS,UAI,LTO,IVR]
    // Return a string with country-specific interpretation
    // Must handle: !hc (no data), low/mid/high thresholds for relevant dimension
    if (!hc) return 'Add your current country to see how this applies.';
    const pdi = hc[0];
    if (pdi > 70) return `In ${cc}, ...`;
    if (pdi > 45) return `In ${cc}, ...`;
    return `In ${cc}, ...`;
  },
  why: 'Root cause explanation — cultural, psychological, or systemic',
  respond: 'Concrete classroom strategy the teacher can try tomorrow',
  research: 'Keywords: pedagogical concepts, research traditions',
  dest_high: 'What changes if moving to a country scoring HIGH on dest_key',
  dest_low: 'What changes if moving to a country scoring LOW on dest_key',
}
```
